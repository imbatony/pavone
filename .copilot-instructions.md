# GitHub Copilot 指令文件

## 🚨 首要规则 - 禁止事项

### ⛔ 绝对禁止

1. **❌ 不要创建任何新的 Markdown 文档文件** (除非用户明确要求)
   - 不要创建 CHANGELOG.md、SUMMARY.md、TODO.md 等
   - 不要创建任何总结性、记录性的文档
   - 完成工作后直接向用户汇报，不要写文档

2. **❌ 不要使用 pip 或 uv pip**
   - 禁止: `pip install`、`uv pip install`
   - 正确: `uv add package_name` (添加依赖)
   - 正确: `uv sync` (同步依赖)

3. **❌ 不要在 tests/ 目录外放置测试文件**
   - 所有测试必须在 `tests/` 文件夹
   - 不要在源代码目录创建测试文件

4. **❌ 不要为简单代码添加冗余注释**
   - 优先用清晰的命名表达意图
   - 只在逻辑复杂时添加中文注释

5. **❌ 不要使用 print()**
   - 用户交互输出: `click.echo()`
   - 错误/警告日志: `logger.error()` / `logger.warning()`
   - 调试信息: `logger.debug()`

## 项目概述

**PAVOne** - Python 视频管理工具 (CLI，插件化架构)
- **语言**: Python 3.10+
- **依赖管理**: uv (pyproject.toml 为唯一配置源)
- **CLI 框架**: Click
- **测试**: pytest
- **类型检查**: Pyright (standard 模式)
- **格式化**: Black + isort
- **Lint**: flake8

---

## 核心规范

### 代码风格
- **格式化**: Black (行长 127)、isort (black profile, 行长 127)
- **Lint**: flake8 (配置在 `.flake8`)
- **类型检查**: Pyright strict 模式，所有新代码必须通过 `uv run pyright pavone/`
- **类型注解**: 强制使用
- **命名**: PascalCase (类)、snake_case (函数)、UPPER_SNAKE_CASE (常量)
- **文档字符串**: 使用中文，仅在必要时添加

### 异常处理
- 所有应用层异常继承 `pavone.core.exceptions.PavoneError`
- 按功能域使用: `NetworkError`, `DownloadError`, `ExtractError`, `PluginError`, `ConfigError`, `MetadataError`
- CLI 入口 (`pavone/cli/__init__.py`) 统一捕获 `PavoneError` 子类
- 退出码使用 `pavone.core.exit_codes.ExitCode` 常量

### 插件架构
所有插件继承 `BasePlugin` 并实现标准接口

### 配置管理
使用 Pydantic 模型定义配置，`pyproject.toml` 为唯一构建配置文件

### 依赖管理 (uv)
```bash
uv sync                    # 同步依赖 (自动安装 dev 依赖)
uv add package_name        # 添加运行时依赖
uv add --dev pytest        # 添加开发依赖
uv run pytest              # 运行测试
```

## 常用命令

```bash
# Windows
.\scripts\dev.ps1 test          # 运行单元测试
.\scripts\dev.ps1 format        # 格式化代码
.\scripts\dev.ps1 check         # 完整检查 (format + lint + type-check + test)
.\scripts\dev.ps1 ci            # 本地模拟 CI

# Linux/macOS
make -f scripts/Makefile test
make -f scripts/Makefile check

# 直接使用 uv
uv run pytest tests/ -v         # 运行测试
uv run pyright pavone/          # 类型检查
uv run black pavone/ tests/     # 格式化
uv run isort pavone/ tests/     # import 排序
```
